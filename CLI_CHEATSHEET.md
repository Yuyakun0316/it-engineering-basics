# コマンドライン（CLI）基本操作メモ

コマンドラインの基本操作を自分なりに整理した備忘録です。

## 📁 ファイル・ディレクトリ操作
- `touch [ファイル名]` : 空のファイルを作成
- `mkdir [ディレクトリ名]` : 新しいディレクトリを作成
- `cat [ファイル名]` : ファイルの中身をターミナルに表示
- `cp [元] [先]` : ファイル・ディレクトリのコピー
- `mv [元] [先]` : 移動、またはファイル名の変更
- `rm [ファイル名]` : ファイルの削除（※ディレクトリは `rm -r`）

## 🗺 ナビゲーション
- `pwd` : カレントディレクトリ（今いる場所）を表示
- `ls` : ディレクトリの中身を表示 (`-a` で隠しファイルも表示)
- `cd [ディレクトリ名]` : ディレクトリを移動
- `cd ..` : 1つ親のディレクトリに移動
- `cd ~` : ホームディレクトリに移動
